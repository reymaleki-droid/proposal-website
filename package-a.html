Package A

WordPress + Ready-Made Theme

A professional web presence deployed in 2–3 weeks using a pre-built WordPress theme. This is the fastest path to having a live product catalog online.

Who This Is For

A business that needs a web presence quickly. Fewer than 100 products. No CRM integration needed. No product management automation. One person manages content occasionally. Budget is the primary constraint.

What Is Included

  • WordPress installation and configuration on client-provided hosting
  • Licensed premium theme selected from 2–3 pre-vetted options (client chooses)
  • Homepage, About, Contact, Product Catalog (single-level categories), Inquiry Form
  • Up to 5 static pages
  • Product entries: up to 100 products from client-provided data
  • Basic SEO configuration (meta tags, sitemap, robots.txt)
  • Contact form with email routing
  • Mobile-responsive layout (theme-dependent)
  • SSL certificate installation
  • 30-day post-launch bug fix window

What Is Excluded

  • Custom design or layout modifications beyond theme options
  • CRM integration
  • Product import automation
  • Multi-language support
  • E-commerce / payment processing
  • Advanced filtering or product comparison features
  • Telegram bot or any automation
  • Ongoing content updates (training provided instead)
  • Performance optimization beyond theme defaults
  • Custom product specification tables

Dependencies

Client must provide the following before or during the project:

  • Hosting credentials and domain access
  • Product data: name, description, image, category, and price for each product
  • Brand logo and guidelines (if any)

Maintenance Model

Client self-manages via WordPress admin. Theme and plugin updates are the client's responsibility. Optional: a monthly maintenance retainer covering updates, backups, and minor fixes can be arranged separately.
