Package C

Custom-Coded Website + CRM

A fully custom product catalog platform with its own admin panel, advanced search and filtering, product comparison, and bi-directional CRM integration. Built from scratch, no WordPress dependency.

Who This Is For

A business with 200+ products, organized product data, an existing or planned CRM, and a desire for a website that functions as a true product catalog platform. Has technical staff or a designated point person for CRM. Plans to scale product offerings over time.

What Is Included

  • Custom front-end (no CMS dependency; modern framework)
  • Custom back-end with admin panel for product, category, and content management
  • Product catalog: multi-level categories, advanced filtering (spec, brand, category, availability), full-text search
  • Product specification tables (per product type schema)
  • Product detail pages: hero image gallery, structured spec table, related products, inquiry form, datasheet PDF link
  • Product comparison interface (user-selected, side-by-side, up to 4 products)
  • CRM integration (bi-directional): inquiries pushed as leads, product interest tracking, pipeline status
  • CRM options: Bitrix24, HubSpot, or Zoho CRM (others evaluated on request)
  • Inquiry form with product context, customer details, routing logic
  • Admin panel: product CRUD, category management, inquiry dashboard, basic analytics
  • SEO: Schema.org Product markup, dynamic sitemap, clean URLs
  • Performance: server-side rendering, image optimization, CDN-ready
  • Security: input validation, rate limiting, HTTPS, admin authentication
  • RESTful API layer for all product and inquiry data
  • Up to 500 initial products entered
  • 90-day post-launch stabilization window

What Is Excluded

  • Telegram bot integration
  • E-commerce / payment processing
  • Multi-language (can be added as extension)
  • Automated product import from supplier feeds (can be scoped as add-on)
  • Mobile application
  • CRM customization (client's CRM must be configured by client or CRM vendor)
  • Ongoing content management (training and admin panel provided)

Dependencies

  • Product data in structured format (spreadsheet or database export with consistent schema)
  • CRM account provisioned with API access
  • Brand assets (logo, colors, fonts)
  • Hosting environment (cloud recommended; specifications provided)
  • Designated client contact for CRM-side decisions

Maintenance Model

Quarterly security patches and dependency updates. Monthly backup verification. CRM integration monitoring (sync health check). Admin panel bug fixes under SLA. Feature additions scoped and priced separately.
