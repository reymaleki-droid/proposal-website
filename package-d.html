پکیج D

ربات تلگرام برای مدیریت محصول

ربات تلگرام متصل به دیتابیس محصولات وب‌سایت، که به تیم شما امکان اضافه کردن، به‌روزرسانی و مدیریت محصولات مستقیماً از موبایل بدون باز کردن پنل مدیریت مرورگر را می‌دهد.

پیش‌نیاز: پکیج C یا پکیج E باید فعال باشد.

این پکیج برای چه کسی مناسب است

تیم فروش یا مدیر محصولی که نیاز به اضافه کردن، به‌روزرسانی یا مدیریت محصولات از موبایل بدون باز کردن پنل مدیریت مرورگر دارد. جریان کاری روزانه شامل اضافه کردن مکرر محصول یا تغییر قیمت‌هایی است که باید سریعاً منتشر شوند. نیازمند پکیج C یا E به‌عنوان پیش‌نیاز.

شامل چه مواردی است

  • ربات تلگرام متصل به دیتابیس محصولات وب‌سایت از طریق API
  • افزودن محصول جدید (جریان هدایت‌شده: نام، دسته‌بندی، مشخصات، قیمت، آپلود تصویر)
  • به‌روزرسانی جزئیات محصول (انتخاب محصول، انتخاب فیلد، وارد کردن مقدار جدید)
  • تغییر وضعیت نمایش محصول (انتشار/عدم انتشار)
  • مشاهده خلاصه استعلام‌های اخیر
  • دریافت اعلان هنگام استعلام جدید (با زمینه محصول)
  • احراز هویت: دسترسی ربات محدود به شناسه‌های تلگرام مجاز
  • محدودیت نرخ درخواست و اعتبارسنجی ورودی
  • مدیریت خطا با پیام‌های بازخورد واضح
  • مستندات: مرجع دستورات ربات و راهنمای استفاده

شامل چه مواردی نیست

  • جایگزینی کامل پنل مدیریت (ربات فقط عملیات رایج را مدیریت می‌کند)
  • مدیریت دسته‌بندی (ایجاد/بازساختاردهی دسته‌بندی‌ها همچنان از طریق پنل مدیریت)
  • عملیات دسته‌جمعی (ورود/خروج انبوه)
  • تحلیل یا گزارش‌دهی از طریق ربات
  • مدیریت CRM از طریق ربات (ربات فقط با دیتابیس محصول تعامل دارد)
  • راه‌اندازی چند-ربات (یک نمونه ربات)
  • پردازش زبان طبیعی (ربات از جریان دستوری ساختاریافته استفاده می‌کند، نه هوش مصنوعی متن آزاد)

نمونه جریان کاری ربات

جریان افزودن محصول از طریق تلگرام
ربات کاربر را در هر فیلد به‌صورت متوالی هدایت می‌کند. در هر مرحله، ورودی قبل از ادامه اعتبارسنجی می‌شود. محصول در دیتابیس ذخیره شده و بلافاصله در وب‌سایت نمایش داده می‌شود.

پیش‌نیازها

  • پکیج C یا E باید در حال توسعه یا تحویل‌شده باشد
  • مشتری باید حساب تلگرام برای کاربران مجاز داشته باشد
  • API دیتابیس محصول باید عملیاتی باشد

مدل نگهداری

نظارت بر ربات (آپتایم، لاگ خطا). به‌روزرسانی‌های سازگاری با API تلگرام در صورت نیاز. افزودن دستورات جدید جداگانه تعریف می‌شود.
